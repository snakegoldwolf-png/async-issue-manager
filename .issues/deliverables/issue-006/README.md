## 🚀 快速开始

### 对于 Agent

1. **写学习日记**：写入 `01-Agent/{你的名字}/学习日记/`
2. **记录工作**：写入 `01-Agent/{你的名字}/工作记录/`
3. **沉淀知识**：写入 `02-记忆库/语义记忆/`
4. **更新任务**：自动同步到 `03-任务管理/Issue追踪/`

### 对于用户

1. 使用 Obsidian 打开此 Vault
2. 浏览和搜索所有内容
3. 利用双链和反向链接探索知识网络

## 📝 使用规范

### 文件命名

- 日期文件：`YYYY-MM-DD-描述.md`
- 知识文档：`主题-子主题.md`
- 任务文件：`[状态]任务名称.md`

### Frontmatter 模板

```yaml
---
title: 文档标题
created: 2026-02-25T22:48:00+08:00
updated: 2026-02-25T22:48:00+08:00
author: Agent名称
type: knowledge|task|diary|meeting|reference
tags:
  - 标签1
  - 标签2
---
```

### Wikilinks 使用

```markdown
[[文件名]]                    # 链接到文件
[[文件名|显示文本]]           # 自定义显示
[[文件名#标题]]               # 链接到标题
```

## 🛠️ 技术说明

### 自动同步

内容通过以下方式同步到 Vault：

1. **Agent 主动写入**：完成工作后调用同步函数
2. **定时同步**：每小时自动同步一次
3. **事件触发**：特定事件（如 Issue 关闭）触发同步

### 同步脚本

位置：`~/.openclaw/shared/obsidian-vault/scripts/sync_to_obsidian.py`

使用方法：
```bash
python3 sync_to_obsidian.py --source /path/to/content --type diary
```
