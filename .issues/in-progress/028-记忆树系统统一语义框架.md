## 背景

bro 的设想：把整个团队的记忆系统设计成一棵真正的树。

## 架构

1. **主树干** = 共有知识库（Obsidian Vault）
2. **分支** = 每个 Agent 的独立 workspace
3. **叶子** = 知识/记忆（置信度决定生命周期）
4. **土壤** = 归档区（养分循环）

## 置信度映射

- >0.8 → 🌿 绿叶茂盛
- 0.5-0.8 → 🍂 叶子变黄
- <0.5 → 🍁 枯萎落叶
- =0 → 🪨 落入土壤

## 需要新增

1. 精华提取（归档时自动提取关键信息）
2. 养分回流（精华同步到 Obsidian）
3. 跨 Agent 搜索

## 前提

- 所有 Agent 必须用索引的方式搜索记忆
- 不能直接读文件，要通过统一的记忆系统
